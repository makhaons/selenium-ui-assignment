`Introduction`
------------

UI automation assignment project for Backbase.

`Requirements`
------------

You need to check Apache Maven & Java 8 is already installed on your computer.

`Running Tests`
-------------

Tests can be triggered by simple mvn command.

Example: `mvn clean install`

You can run with right click `MainRunner.java` class which is in java folder as well.

You may find a test report under `target/test-report/report.html` after the test run completed.

You may find manual test cases on the root path.

`Technologies`
------------

- Java 8
- Maven
- Selenium
- Cucumber
- Bonigarcia (WebDriver Manager)
- ExtentReport
